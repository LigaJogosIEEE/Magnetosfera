# Mecânicas do jogo

## Movimentação

Mover-se para direita e esquerda, com intuito de não cair no vácuo sem fim. A forma de se movimentar é tocando:

* Esquerda da Tela: Move-se para a esquerda
* Direita da Tela: Move-se para a direita
